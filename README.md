# Todo List MERN application

## Usage

add .env file to add your MONGO_URI

- MONGO_URI = <your mongodb url>
- NODE_ENV = development || production
- PORT = 8000
- JWT_SECRET = <your jwt token>
### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd view
npm install
```

### Run Server

```
npm run server
```
